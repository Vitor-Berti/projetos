# 💻 Agenda 11 - Controle de Níveis de Água

Este projeto em python teve como objetivo praticar estruturar como: **listas, funções e bibliotecas**.
O programa utiliza funções e a biblioteca de colometria para que possa ser criado um sistema de níveis e cores respectivas de cada nível. O programa solicita qual o nível do reservatório de água, e mostra em qual situação está e sua cor correspondente.
