# ⚡Projeto: Consumo de energia

Esse projeto em python foi criado para a utilização de: **Entrada -> Processamento -> Saída**
O programa solicita qual o aparelho eletrônico utilizado, sua potência em Watts e o tempo médio de horas utilizadas por dia.
Após isso, calcula o consumo mensal utilizando **consumomensal  = (potencia * horasdia * 30)/ 1000**
